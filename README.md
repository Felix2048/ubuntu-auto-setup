# ubuntu-auto-setup

An auto-setup script for Ubuntu:

- 16.04 Desktop (x86_64)
- 18.04 Desktop (x86_64)

## Usage

- Configure your profile:
    ```bash
        vim ./config/setupConfig
    ```
- Launch the script with:
    ```bash
        ./Setup
    ```
